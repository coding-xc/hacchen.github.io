TypeScript

TS简介

Ts是什么

  1. 以JavaScript为基础构建的语言
  2. 一个Javascript的超集
  3. 可以在任何支持JavaScript的平台中执行
  4. TypeScript扩张了Javascript并添加了类型
  5. TS不能被JS解析器直接执行

TS增加了什么

  1. 类型
  2. 支持ES新特性
  3. 添加了ES不具有的新特性
  4. 丰富的配置选项
  5. 强大的开发工具

基本类型

类型声明

  1. 类型声明是TS非常重要的一个特点

  2. 通过类型声明可以指定TS中变量(参数、形参)的类型

  3. 通过指定类型后，当变量赋值时，TS编译器会自动检查值是否符合类型声明，符合则赋值，否则报错

  4. 简言之，类型声明给变量设置了类型，使得变量只能存储某种类型的值

  5. 语法

webpack打包

​ npm i -y 下载package.json文件

​ npm i -D webpack webpack-cli typescript ts-loader

​ npm i -D html-webpack-plugin 此插件可在webpack中配置，自动生成html文件

​ npm i -D webpack-dev-server webpack 开发服务器，根据项目改变自动刷新

​ npm i -D clean-webpack-plugin 清除dist目录

​ npm i -D @babel/core @babel/preset-env babel-loader core-js 解决兼容性问题
